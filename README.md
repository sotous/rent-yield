# rent-yield

Monorepo for a fullstack real estate application focused on exploring sale price, rent, and investment yield signals.

## Vision

`rent-yield` helps users evaluate residential properties by comparing sale price against rental income potential through clear visual exploration and decision-friendly metrics.

## Monorepo Structure

- `apps/frontend`: frontend application
- `apps/backend`: backend application
- `specs`: product and technical specifications

## Current Focus

We are defining the product through spec-driven development, starting with the frontend use cases and the core real estate investment metrics.
